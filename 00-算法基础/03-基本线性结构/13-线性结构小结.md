# 线性结构小结

## 线性结构

- 线性数据结构 Linear DS 将数据项以某种线性的次序组织起来
- 栈 Stack 维持了数据项后进先出 LIFO 的次序
- 队列 Queue 维持了数据项先进先出 FIFO 的次序
- 双端队列 Deque 可以同时具备栈和队列的功能
- 列表 List 是数据项能够维持相对位置的数据集
- 链表可以保持列表维持相对位置，不需要连续的存储空间
- 链表实现无序表或者有序表时，其各种方法，对链表头部 head 需要特别的处理
